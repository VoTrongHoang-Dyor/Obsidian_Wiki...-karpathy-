# LLM Overview Sample

## Key Points
- LLMs are trained on massive text datasets to predict next tokens
- Transformer architecture is dominant (Vaswani et al. 2017)
- Capabilities include text generation, summarization, code writing
- Limitations: hallucinations, context window constraints, no true reasoning

## Sources Referenced
- Vaswani et al. (2017) Attention Is All You Need
- OpenAI GPT-4 Technical Report (2023)
